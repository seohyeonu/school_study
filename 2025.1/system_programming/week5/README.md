# I. scoreProsseing

- 성적 처리 프로그램이다.
- 성적처리 프로그램은 아래의 기능을 수행한다.
  1. 성적 입력
  2. 성적 수정
  3. 성적 출력

## 1. 성적입력

1. 성적입력은 메인 메뉴에서 1을 입력하여, 입력모드로 진입하여 시작한다.
2. 입력모드로 진입 후 학번을 입력하게된다. 학번은 아래의 규칙을 지켜야한다.
   1. 학번은 숫자로 이루어진 8글자이다.
   2. 학번에 숫자외 다른 문자는 입려되면 안된다.
   3. 0을 입력하면 입력 모드는 종료된다.
3. 학번 입력후 성적을 입력한다.
4. 성적 입력은 아래와 같은 규칙을 따른다.
   1. 국어, 영어, 수학, 사회, 과학 순으로 입력하되 각 성적은 띄어쓰기로 구분된다.
   2. 성적의 입력 범위는 0~100사이의 정수이다.
5. 입력된 성적을 확인하고 사용자의 입력에 따라 성적을 레코드에 저장한다.


## 2. 성적 수정

- 성적 수정은 학번으로 검색 후 성적 전체를 재입력하거나 특정 과목만 수정하는 방식으로 동작
- 성적 삭제는 레코드를 지우는 방식으로 동작


## 3. 성적 출력

- 성적 출력 방법은 아래의 4가지 방법으로 이루어 진다.
  1. 입력된 성적 전체를 보여주는 방법
  2. 입력된 과목에 맞제 출력하는 방법
  3. 학번 검색 가능
  4. 학번 + 입력된 과목 검색 가능





# II. scoreAnalyze

이 프로그램은 `Score.bat` 파일에서 학생 성적 레코드를 읽어와 통계 및 등급을 계산하고 출력합니다.  

## 주요 기능

1. **원본 파일 복제**  
   - `Score.bat` → `Score.bak` (임시 작업용)  
2. **레코드 파싱**  
   - 고정 길이(23바이트) 레코드:  
     - 학번(8자리) + 국어·영어·수학·사회·과학(각 3바이트)  
3. **통계 계산**  
   - 전체 학생 평균 점수  
   - 과목별 평균 점수  
   - 전체 과목 통합 최고/최저 점수  
   - 과목별 점수 분포(0–9, 10–19, …, 90–99, 100)  
4. **등급 부여**  
   - 각 과목별 상위 30%: A, 30–70%: B, 70–90%: C, 90–95%: D, 나머지: F  
5. **결과 출력**  
   - 통계 및 각 학생·과목별 등급을 터미널에 출력  

## 파일 구조

- **main.c**  
  메인 로직. 파일 복제, 레코드 읽기, 통계/분포/등급 계산, 결과 출력.  
- **Score.bat**  
  원본 성적 레코드(8+5×3바이트씩).  
- **Score.bak**  
  임시 복제 파일(읽기/쓰기 모두 사용).
